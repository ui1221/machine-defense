# 実装プラン

## 1. 周回報酬を入れる

戦闘終了時に `クレジット` を獲得できるようにする。

- 勝利時は多めに獲得
- 敗北時も少し獲得
- 「遊べば必ず何か進む」状態を作る

実装状況: 完了

## 2. 拠点に強化タブを追加

クレジットを使って恒久強化できる画面を作る。

最初は以下だけを実装する。

- アサルト型 攻撃力Lv
- アサルト型 冷却Lv
- バリケード耐久Lv
- 装備強化Lv

実装状況: 完了

## 3. 恒久強化を戦闘に反映

戦闘開始時に、拠点強化の効果をキャラやバリケードへ反映する。

これで以下の基本ループを完成させる。

```text
戦闘する → 報酬を得る → 強化する → 少し先に進める → 詰まったら周回する
```

実装状況: 完了

## 4. ステージ進行ロック

クリアしたステージの次だけ解放する。

- 未解放ステージは表示する
- 未解放ステージは選択不可
- 30ステージの先が見える形にする
- デバッグ用に全ステージ解放ボタンを用意する

実装状況: 完了

## 5. 30ステージ化

既存ステージをベースに、データで30ステージまで増やす。

- 各ステージは5分前後
- 序盤は基本型中心
- 中盤から高速・装甲・群体を増やす
- 後半で精鋭型や質量型を増やす

実装状況: 完了

## 6. 周回の旨味を増やす

ステージごとに報酬差を作る。

- 低難度は安定してクレジット
- 中盤以降は装備ドロップ率や素材量が増える
- 短時間周回と高難度周回の役割を分ける

実装状況: 完了

## 7. 敵特殊能力を追加

強化システムが入ってから、敵に個性を追加する。

最初の候補:

- 装甲型: 被ダメージ軽減
- 加速型: 途中から加速
- 搬送型: 撃破時に小型敵を出す
- 破砕型: バリケード攻撃前に警告演出
- 精鋭型: 複合能力

実装状況: 完了

## 8. 装備システムを育てる

装備強化、レア度、ステージ別ドロップなどを追加する。

ここまで来ると、周回目的がかなり強くなる。

実装状況: 完了
