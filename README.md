# 🧟 DEAD DEFENCE

上から押し寄せるゾンビの群れからバリケードを守るローグライト系タワーディフェンスゲームです。  
レベルアップのたびに強化を選択し、戦闘を重ねながら独自のビルドを育てていきましょう。

---

## 🎮 ゲーム概要

- **ジャンル**: ローグライト タワーディフェンス
- **視点**: 縦スクロール（敵が画面上から下へ向かって迫ってくる）
- **目的**: 全ての敵を撃破し、バリケード（HP）を守り切る
- **技術スタック**: [Phaser 3](https://phaser.io/) + TypeScript + Vite

---

## 🕹️ 操作方法

| 操作 | 内容 |
|------|------|
| タップ / クリック | キャラクターの攻撃方向を指定（インプット送信） |
| レベルアップ時 | カードをタップして強化を選択 |

---

## 📺 シーン構成

```
BootScene（起動） → HomeScene（拠点） → BattleScene（戦闘） → ResultScene（リザルト）
                                                ↕
                                        BattleUIScene（戦闘HUD・重畳）
```

### 🏠 HomeScene（拠点）
3つのタブで構成されています。

| タブ | 内容 |
|------|------|
| ⚔️ ステージ | ステージを選択して戦闘へ（クリア済みは ✅ 表示） |
| 🧑 キャラ | キャラクター一覧と基本ステータスの確認 |
| 🎒 装備 | 入手した武器をキャラクターに装備（1キャラ1武器） |

### ⚔️ BattleScene（戦闘）
メインのゲームプレイシーン。スポーン管理・攻撃処理・当たり判定・レベルアップを担当します。

### 🖥️ BattleUIScene（戦闘HUD）
BattleScene に重畳表示されるUIレイヤー。EXPバー・レベル・バリケードHP、レベルアップ選択画面を表示します。

### 🏆 ResultScene（リザルト）
勝利/敗北の判定・統計表示・ドロップ武器の保存・リトライボタンを担当します。

## 🧑 キャラクター

| キャラ | 攻撃タイプ | 特徴 |
|--------|-----------|------|
| 🧑 ソルジャー | `bullet` | バランス型。攻撃力・速度ともに標準 |
| 🎯 スナイパー | `bullet` | 超高火力の貫通弾。クールタイムが長い |
| 🧝 エルフ | `burst` | 高速連射。強化で同時発射数が増える |
| 🛡️ ナイト | `slash` | 遠距離斬撃。高威力・ノックバック付き |
| 🧙 メイジ | `mage` | 着弾地点に炎床を生成。範囲内は鈍足 |

初期キャラクターは **ソルジャー** 固定（ステージ設定で変更可能）。  
最大 **5体** まで同時配置できます。

## ⬆️ レベルアップ・強化システム

敵を倒すと経験値を獲得し、バーが満タンになるとレベルアップします。  
レベルアップ時に **3択のランダム強化** が提示されます（ゲームが一時停止）。

### 強化カテゴリ

| カテゴリ | 内容 |
|---------|------|
| `add_character` | 新キャラクターを仲間に加える（最大5体） |
| `stat_boost` | 攻撃力・攻撃速度をパーセント強化（最大スタック数あり） |
| `special` | 貫通、バースト増加など特殊効果を付与 |

### レア強化（Lv.3以降解放）

| 強化名 | 条件 | 効果 |
|--------|------|------|
| 🌟 ソルジャー覚醒 | ソルジャー在籍 & Lv.3以上 | 攻撃力 +50% & 攻撃間隔 -30% |
| 🌟 スナイパー 必殺の一撃 | スナイパー在籍 & Lv.3以上 | 攻撃力 +100% |
| 🌟 エルフ 矢の嵐 | エルフ在籍 & Lv.3以上 | 同時発射 +2 & 攻撃間隔 -20% |
| 🌟 ナイト 烈風斬 | ナイト在籍 & Lv.3以上 | 攻撃力 +60% & 攻撃間隔 -25% |
| 🌟 メイジ 業火 | メイジ在籍 & Lv.3以上 | 攻撃力 +70% & 攻撃間隔 -20% |
| 🌟 全軍鼓舞 | 3体以上在籍 & Lv.4以上 | 全員 攻撃力 +25%（最大2回） |
| 🌟 全軍加速 | 3体以上在籍 & Lv.4以上 | 全員 攻撃間隔 -20%（最大2回） |

## 🗡️ 武器システム

敵を倒したとき **0.1%** の確率で武器がドロップします。  
入手した武器はホーム画面の「🎒 装備」タブからキャラクターに装備できます（1キャラ1武器）。

| 武器 | 効果 |
|------|------|
| 🗡️ ナイフ | 攻撃力 +15% |
| ⚔️ 鉄の剣 | 攻撃力 +30% |
| 🏹 弓 | 射程 +25% |
| 🪄 魔法の杖 | 攻撃力 +20% & 攻撃速度 +10% |
| 🔫 拳銃 | 攻撃速度 +25% |
| 🗡️ 大剣 | 攻撃力 +50% / 攻撃速度 -20% |

武器の効果は戦闘開始時にキャラクターのステータスへ乗算で適用されます。

## 💾 セーブデータ

`localStorage` の `td_save` キーに JSON 形式で保存されます。

```typescript
interface GameSave {
  clearedStages: string[]   // クリアしたステージIDの配列
  ownedWeapons: OwnedWeapon[] // 入手済み武器と装備先キャラ
}
```

```

    ├── objects/
    │   ├── Character.ts     # キャラクターGameObject
    │   ├── Enemy.ts         # 敵GameObject
    │   ├── Bullet.ts        # 弾GameObject
    │   ├── Barricade.ts     # バリケードGameObject

    ├── systems/
    │   ├── SpawnManager.ts  # 敵スポーンタイミング管理
    │   ├── InputManager.ts  # タップ/クリック入力管理
    │   ├── TargetingSystem.ts # 敵のターゲティングロジック

        ├── HomeScene.ts     # 拠点・ステージ選択・装備管理
        ├── BattleScene.ts   # メインゲームプレイ

        └── ResultScene.ts   # リザルト表示・セーブ
```

## 🚀 セットアップ & 実行

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# プロダクションビルド
npm run build
```

開発サーバー起動後、ブラウザで `http://localhost:5173` を開いてください。
